# KYL.DEV — Vibe Coder Portfolio

> A cinematic, immersive portfolio for **Jayson Ponio Mallari** (A.K.A **Kyl Ponio**).
> Cyberpunk × hologram aesthetic. Next-gen gaming lobby meets cyberpunk dev workspace.

---

## Quick Pitch
- 3D futuristic room as the **landing experience** — not a hero image, a hero *world*
- A hologram **chibi character** of Kyl rotates in the center like a "character select"
- Interactive **monitors** show scrolling code, a terminal, and a **playable retro mini-game**
- An interactive **glowing guitar** triggers a cinematic "walk → sit → play" sequence
- Scroll-driven cinematic transitions through About → Projects → Contact
