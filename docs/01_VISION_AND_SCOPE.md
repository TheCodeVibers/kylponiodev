# 01 — Vision & Scope

## Identity
**Jayson Ponio Mallari** — A.K.A **Kyl Ponio** — **Vibe Coder**

Vibe Coding is the act of building immersive, soulful interactive experiences where **craft, sound, motion, and code converge**. It's where engineering meets play. It's about *feel* as much as function.

---

## Brand Pillars
1. **Cinematic** — every screen feels like a scene from a sci-fi film
2. **Immersive** — 3D worlds, ambient sound, parallax, depth
3. **Playful** — game-loop interactions; everything clickable rewards the click
4. **Crafted** — small details: scanlines, particles, ambient hums, glow falloffs

## Audiences

- **Conference / talk organizers** looking for speakers in interactive web
- **Clients** for freelance immersive web projects

---

## Goals
1. **Stop the scroll within 2 seconds** — the hero must hit
2. **Communicate range** (fullstack + 3D + game + music) within 1 minute
3. **Make the visitor want to *play* with the site**, not just read it
4. **Land a follow-up action**: GitHub follow, contact-form submit, or social DM

---

## Scope — V1

### In Scope
- 3D landing room with rotating chibi hologram character
- Holographic name + title overhead
- Floating particles + scanlines + bloom glow
- Mouse-parallax camera with subtle head-tracking
- Desk with 3 animated monitors (code, terminal, game)
- Click-to-expand fullscreen playable mini-game
- Click-to-trigger guitar cinematic (walk → sit → play)
- Scroll-driven About + Projects + Contact sections
- 3 project cards: **Chat System**, **Admin Dashboard**, **Frontend Developer Projects**
- Mobile-responsive lightweight fallback
- Audio toggle + reduced-motion support

### Out of Scope (Future)
- CMS / blog
- Multi-language
- Authenticated dashboard
- Multiplayer / WebRTC interactions
- AR / VR mode
- Custom 3D model marketplace

## Non-Goals (Things We Intentionally Do *Not* Optimize For)
- Pixel-perfect IE11 / legacy browser support
- SEO ranking for generic keywords (this is a portfolio, not content marketing)
- Print stylesheets
- Server-rendered 3D fallbacks (we accept that no-JS means a static hero only)
